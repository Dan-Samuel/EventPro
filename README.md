# 🎉 EventPro

EventPro is a modern event discovery and management web application that allows users to browse events, register for events, and for organizers to create and manage their own events.

Built using HTML, CSS, and Vanilla JavaScript with a mobile-first, responsive design.

---

## 🚀 Features

### 👥 Users can:
- Browse featured and upcoming events
- Search and filter events by category
- View full event details
- Register for events
- View tickets (upcoming and past events)

### 🧑‍💼 Organizers can:
- Create new events
- Access a dashboard
- View event overview stats
- Manage their events

---

## 🎨 UI Features
- Mobile-first responsive design
- Clean dashboard layout with tabs

## ⚙️ How It Works

EventPro uses hash-based routing:

- #/ → Landing page  
- #/events → Browse events  
- #/event → Event details  
- #/register → Register for event  
- #/dashboard → Organizer dashboard  
- #/create → Create event  
- #/tickets → My tickets  
- #/signin / #/signup → Authentication pages  

JavaScript handles page switching without reloading the browser.
